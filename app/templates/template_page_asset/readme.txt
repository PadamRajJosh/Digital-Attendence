Why this folder template_page_assets?
This folder consists of the pages/templates used in the project.


How to add new page/view.
    Basically use the html page that suits for particular usage.

    blank.html is most prefered. Extra modification can be applied to the pages using Bootstrap
    to display many user interactive GUIs


Caution: While editing a template. Be careful to know which div correspons to which GUI components.


Note: Other style assets can be found inside the static_folder of the project. But do not modify them.
